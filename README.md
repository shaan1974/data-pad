# VueJS Data-Pad

**Purpose :**

As i need a date picker to replace the input date into the filter component of Data-greep, i'v build this one.

**Dependecies :**

It use moment.js

**Options: **

```
"PadConfig":
    {
        "firstDayOfTheWeekMonday": true,
        "showCurrentDay": true,
        "css":
        {
            "right_arrow": "",
            "left_arrow": ""
        },
        "format": "DD-MM-YYYY"
    }
```
